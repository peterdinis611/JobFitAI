# Compare analyses

Route: `/compare`

Side-by-side comparison of two saved analyses — same or different roles.

## From the dashboard

1. On **History**, check two rows
2. Click **Compare selected**

Query params `?a=<id>&b=<id>` pre-fill the selectors.

## Manual selection

Use the dropdowns to pick any two analyses from your history.

## Comparison view

For each analysis:

- Match score ring
- Seniority fit
- Top matching / missing skills (up to 8 each)
- Link to full report

A **delta banner** shows the point difference (e.g. "+12 points higher on the right").

## Use cases

- Same CV, two different companies
- Before/after CV revision (also see re-score delta on the new report)
- Choosing which role to prioritize
