# Your first analysis

End-to-end walkthrough from zero to a saved match report.

## 1. Create an account

Sign up with email and password. You stay signed in via Convex Auth.

## 2. Upload a resume

1. Go to **Resumes** in the nav
2. Drag & drop a **PDF** or **DOCX** (max 10 MB), or click **Upload CV**
3. The latest upload becomes your **active** resume automatically

:::tip Preview
After the first analysis, open **Preview** on a resume card to see parsed text and word count.
:::

## 3. Run a match

1. Go to **Analyze**
2. Confirm your active resume is shown
3. Choose **Paste text** or **URL**:
   - **Text** — paste the full job description
   - **URL** — HTTPS link only (e.g. company careers page)
4. Click **Run analysis**

Watch the **Live agent stream** — you'll see tool calls: `parse_resume` → `score_match` → `save_analysis` (and fetch steps for URLs).

## 4. Read the report

When complete, open the report from **History** (dashboard). You'll see:

- Match % ring and progress bar
- Skill radar chart (when categories are available)
- Matching / missing skills
- Red flags and recommendations

## 5. Go further

| Goal | Where |
|------|-------|
| Tailor CV bullets | Report → **Career tools** → Tailor bullets |
| Draft cover letter | Report → **Cover letter** |
| Close skill gaps | Report → **Learning plan** |
| Track application | Report → **Save to tracker** |
| Compare two roles | History → select two rows → **Compare selected** |
| Re-score after CV edit | Report → **Re-score with new CV** |

## Example job text

If you don't have a posting handy, paste any role description with clear requirements (skills, years of experience, responsibilities). The agent extracts requirements and scores honestly — vague postings yield less precise results.
