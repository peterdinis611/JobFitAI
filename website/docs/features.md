# Features

## Core analysis

- Paste job description or provide an HTTPS URL
- Live agent stream with tool calls
- Match score ring, skill radar chart, red flags, recommendations
- Export report as Markdown (copy to clipboard)
- Compare two analyses side-by-side

## Career tools (per report)

Available on each analysis detail page under **Career tools**:

| Feature | Description |
|---------|-------------|
| **Tailor bullets** | Agent rewrites 3–5 resume bullets with before/after view |
| **Cover letter** | One-click draft from resume + job + gap context |
| **Learning plan** | 2-week actionable plan per missing skill |
| **Re-score** | Upload revised CV, new analysis linked with % delta |
| **Save to tracker** | Add role to application kanban |

## Application tracker

Route: `/tracker`

Kanban columns: **Saved → Applied → Interview → Offer**

Each card links to the underlying match report and shows the match percentage.

## Dashboard

- Stat cards (total, average, best score)
- Filter by minimum match %
- Job title on each row (auto-extracted from posting)
- Select two rows to compare

## Resumes

- PDF/DOCX upload (max 10 MB)
- Version history with active resume flag
- Preview parsed text + word count (after first parse)
