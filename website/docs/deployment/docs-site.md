# Documentation site

This documentation is built with [Docusaurus 3.10](https://docusaurus.io/) in `website/`.

## Structure

```
website/
├── docs/                 # Markdown/MDX pages (you are here)
├── docusaurus.config.ts  # baseUrl: /docs/
├── sidebars.ts           # Sidebar navigation
└── src/css/custom.css    # Brand colors
```

## Commands

| Command | Purpose |
|---------|---------|
| `npm run build:docs` | Build → `public/docs/` |
| `npm run dev:docs` | Live reload on port 3001 |
| `npm run build` | Docs + Next.js production build |

## Adding a page

1. Create `website/docs/<section>/<page>.md`
2. Add `_category_.json` if new section
3. Update `website/sidebars.ts` if not autogenerated
4. Run `npm run build:docs`

## MDX features

- [Admonitions](https://docusaurus.io/docs/markdown-features/admonitions): `:::tip`, `:::warning`, `:::info`
- [Mermaid diagrams](https://docusaurus.io/docs/markdown-features/diagrams): ` ```mermaid ` blocks
- Code blocks with syntax highlighting

## Next.js integration

`next.config.ts` rewrites `/docs/*` paths to serve Docusaurus `index.html` files from `public/docs/` (Next.js doesn't resolve directory indexes automatically).

Nav **Docs** link uses a plain `<a href="/docs">` for full-page static load.

## Search

Local full-text search is enabled via `@easyops-cn/docusaurus-search-local` — no Algolia API key required.
